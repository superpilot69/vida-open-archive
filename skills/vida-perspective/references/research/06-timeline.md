# Vida 调研 06: 时间线

## 时间线概览

| 时间 | 事件 | 对思维的影响 |
|------|------|--------------|
| 2020-11 | 发表 `好胜心`、`谈新时代消费主义` 等早期专栏 | 早期就显露出竞争心、反消费主义和反传统路径倾向 |
| 2021-02 | 连续写宏观与市场周期判断，如 `如何看待2022年金融危机？`、`对2021-2022年股票市场的预测` | 流动性与周期框架成型，开始把宏观与资产操作绑定 |
| 2021-04 | 发表 `我的致富思维` | 系统总结冒险、独立、延迟满足、合作、风控等内核 |
| 2021-06 | 写 `有1000万怎么理财` | 明确“保留流动性等待熊市”的资金观 |
| 2021-08 | 发表 `我爸的A股投资思维` | 公开承认父辈价值投资、逆向与安全边际对自己的影响 |
| 2022-08 | 高赞回答 `关于创业有什么建议？` | “轻资产信息差套利 + 小额高频试错”成为对外最核心的方法论 |
| 2022-09 | 发表 `为什么我要开这个知乎账户？` | 把个人 IP、名、利、未来基金叙事正式写明 |
| 2023-10 至 2023-12 | 出现 `如果有条件，你最想去哪个国家生活？`、`高质量英文CT信息源列表`、`币圈到底是谁一直在赚钱？` | 全球化、英文社区、crypto operator 视角成熟 |
| 2024-05 至 2024-09 | 高赞回答 `普通人怎样才能一步步走向成功？`、专栏 `给00后们的话`、`写给黑粉的话` | 从单纯赚钱框架扩展到青年叙事、社会批判与自我品牌维护 |
| 2025-01 | 发表 `年轻真好...`、`2025我的1000万刀美股战略布局` | 资金体量与视角进一步全球化，开始更明确地从 crypto 向 tradfi 配置转移 |
| 2025-02 | 回答 `如何看待bybit被黑客攻击盗取15亿美元资产` | 风险认知更现实，承认 crypto 体系不成熟，强调出金与迁移 |

## 分阶段观察

### 2020: 早期人格底色

- 代表内容：
  - `好胜心`
  - `谈新时代消费主义`
- 关键词：
  - 竞争
  - 反消费主义
  - 反固定路线
- 判断：
  - 这一阶段更像价值观铺底，还没完全形成后来的系统交易/套利公开体系

### 2021: 宏观与资产框架爆发

- 这是仓库内内容最密集的一年，共 303 篇
- 关键词：
  - 周期
  - 流动性
  - A 股 / 港股 / 美股 / crypto
  - 风险与等待
- 代表内容：
  - `如何看待2022年金融危机？`
  - `有1000万怎么理财`
  - `我爸的A股投资思维`
  - `我的致富思维`
- 判断：
  - 这是其公开思维结构真正成型的主阶段

### 2022: 创业方法论定型

- 内容量下降，但代表作更集中
- 代表内容：
  - `关于创业有什么建议？`
  - `为什么我要开这个知乎账户？`
- 判断：
  - 从纯交易/宏观走向“如何复制我这条路径”的公开表达

### 2023: 英文社区和 crypto operator 视角强化

- 代表内容：
  - `高质量英文CT信息源列表`
  - `币圈到底是谁一直在赚钱？`
  - `如果有条件，你最想去哪个国家生活？`
- 判断：
  - 他的世界观从“中国互联网赚钱”扩展到“全球信息流 + 地理套利 + crypto 生态位”

### 2024: 青年叙事与人格对抗性增强

- 代表内容：
  - `普通人怎样才能一步步走向成功？`
  - `给00后们的话`
  - `写给黑粉的话`
- 判断：
  - 这一年不只是讲怎么赚钱，还更明显地讲“你应该成为什么样的人”
  - 同时，黑粉与公众争议开始被纳入品牌结构

### 2025: 资产全球化与风险再评估

- 代表内容：
  - `年轻真好，具有无限的可能。真的就有无限的可能吗？`
  - `2025我的1000万刀美股战略布局`
  - `如何看待bybit被黑客攻击盗取15亿美元资产`
- 判断：
  - 心智模型延续，但容器变化：
    - 从 crypto 赚钱 → tradfi 与全球资产配置
    - 从“泰国也不错” → “迪拜是更优定居地”

## 最近 12 个月动态

- `2024-06-24`：`给00后们的话`
  - 强化“年轻人仍有机会”的公开叙事
- `2024-09-12`：`写给黑粉的话`
  - 高度公开化地处理质疑与人设问题
- `2025-01-13`：`去泰国或者东南亚国家旅行还安全吗？你还敢去吗？`
  - 说明自己在泰国定居过一年，并给出本地判断
- `2025-01-14`：`年轻真好，具有无限的可能。真的就有无限的可能吗？`
  - 再次强调年轻人的时间、合规宽容度和试错优势
- `2025-01-24`：`2025我的1000万刀美股战略布局`
  - 明确 1000 万美元级别的美股配置框架
- `2025-02-22`：`如何看待bybit被黑客攻击盗取15亿美元资产`
  - 公开披露自己在 Bybit 的资金状态和后续风险判断

## 时间线里的核心演化

- 早期的核心矛盾是“反传统”与“想赢”
- 中期的核心矛盾是“赚快钱的 operator”与“长期资产配置者”
- 后期的核心矛盾是“高调个人 IP”与“想进入更高层级资本游戏”
