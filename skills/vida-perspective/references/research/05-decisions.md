# Vida 调研 05: 决策记录与行动

## 可验证的重要决策

### 决策 1: 用轻资产套利替代重资产创业

- 背景：
  - 他把年轻人第一次创业的高失败率视为既定事实
- 决策：
  - 优先做轻资产、低成本、低沉没成本的套利型生意
  - 反对一开始就贷款开店、开厂、搞重资产
- 来源：
  - `关于创业有什么建议？`
  - `年轻人创业失败的主要原因我看到...`（2024-09-15，想法，https://www.zhihu.com/pin/1818660471491919872）
- 暴露出的判断规则：
  - 先保留活下来的机会，再追求放大
- 可信度：高

### 决策 2: 在高估值阶段宁可拿着流动性，也不追高

- 背景：
  - 2021 年他判断流动性宽松尾声将导致未来资产承压
- 决策：
  - 先把大额现金停在高流动性低风险资产上
  - 等待 2022-2023 熊市再部署股票、基金、crypto
- 来源：
  - `如何看待2022年金融危机？`
  - `有1000万怎么理财`
  - `手里有四千万怎么稳定利益最大化？`
- 暴露出的判断规则：
  - 错过短期上涨比在高位接盘更能接受
- 可信度：高

### 决策 3: 公开打造个人 IP，而不是隐藏自己

- 背景：
  - 他把“名”视作权益、影响力和未来募资工具
- 决策：
  - 有意识地运营知乎账户
  - 用公开表达记录财富增长和路径
  - 把高调展示视作吸引资源的一部分
- 来源：
  - `为什么我要开这个知乎账户？`
  - `写给黑粉的话`
- 暴露出的判断规则：
  - 若名气能变成真实商业与关系杠杆，那就值得经营
- 可信度：高

### 决策 4: 在消费上极度克制，把现金继续导入生息资产

- 背景：
  - 他长期把消费等同于损失未来利润
- 决策：
  - 控制月花销比例
  - 大消费要与“大赚一笔”绑定
  - 维持较高现金与资产再投入能力
- 来源：
  - `我的致富思维`
  - `普通人怎样才能一步步走向成功？`
  - `成功的套利者都是反消费主义的拥...`
- 暴露出的判断规则：
  - 钱首先是弹药，而不是奖励
- 可信度：高

### 决策 5: 地理与制度迁移

- 背景：
  - 他多次表达对中国式规训、税制、监管和身份限制的不满
- 决策：
  - 长居泰国
  - 2025 年将理想定居地更新为迪拜
  - 把自己定位成“世界公民 / 国际资本家”
- 来源：
  - `如果有条件，你最想去哪个国家生活？`
  - `去泰国或者东南亚国家旅行还安全吗？你还敢去吗？`
  - `写给黑粉的话`
- 暴露出的判断规则：
  - 把国家和城市也视作可被选择的资产配置项
- 可信度：高

### 决策 6: 从均衡配置转向更进攻的集中押注

- 背景：
  - 早期自认偏 Dalio 式均衡，后期认为那只会得到平均回报
- 决策：
  - 2025 年把 1000 万美元美股头寸转成更集中的进攻型配置
  - 重点押注长债与 Cloudflare
- 来源：
  - `2025我的1000万刀美股战略布局`
- 暴露出的判断规则：
  - 当认知与赔率足够倾斜时，会愿意集中下注
- 可信度：高

### 决策 7: 接受 crypto 的高收益，也在事件后主动降低依赖

- 背景：
  - 长期在 crypto 里做新闻交易、套利、资金费率策略
  - 2025-02 遭遇 Bybit 被黑事件
- 决策：
  - 事发时尝试提现
  - 事后明确说要加快从币圈向传统金融市场出金
- 来源：
  - `币圈到底是谁一直在赚钱？`
  - `如何看待bybit被黑客攻击盗取15亿美元资产`
- 暴露出的判断规则：
  - 只要环境仍有 alpha，他会留在里面
  - 一旦系统性成熟度不足威胁到资金安全，会切换载体
- 可信度：高

## 言行一致与不一致

### 一致的地方

- 反复强调流动性，自己的大额配置也围绕流动性与机会窗口
- 反复强调信息差，自己的学习来源确实高度英文和全球化
- 反复强调 IP 杠杆，自己确实在公开经营 IP
- 反复强调消费克制，公开文本里长期保持这一逻辑

### 有张力的地方

- 他反对满仓梭哈，但 2025 开始允许更集中的进攻仓位
- 他反对“带陌生人赚钱”，但又持续公开展示自己的赚钱能力
- 他强调诚信与不割韭菜，同时又频繁赞美灰度套利和规则边缘机会

## 结论

- Vida 的决策逻辑不是稳健传统派，也不是纯赌博派
- 更准确地说，他是：
  - 高度机会主义
  - 高度流动性导向
  - 低道德说教
  - 但又极其在意赔率与撤退通道
