# Vida Exhibit Web App

This is the source code for the public Vida exhibition website:

- https://vida-exhibit.pages.dev/

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Local Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Data Source

The app reads from:

- `src/data/articles.json`

That dataset is generated from the archived Vida corpus included at the repository root.
